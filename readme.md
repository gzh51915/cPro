## 项目目录说明

### cms 后台管理系统

* public 服务器根目录
* src 源码
    * api axios的二次封装
    * components 组件文件
    * routers 路由懒加载
    * store redux数据
    * utils 工具包
    * view 页面

### cPro 移动端App

* public 服务器根目录
* src 源码
    * api axios的二次封装
    * components 组件文件
    * assets 静态资源

### SF-admin-server

* modules 模型创建文件
* routers 接口文件
* utils 公共包
* server.js 入口文件
